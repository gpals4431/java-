이벤트(Event) - 이벤트 전달

브라우저에서 이벤트가 발생되면 window 객체에 먼저 도착하고 DOMTree를 이용하여 이벤트가 발생된 타겟 태그에 도착한 후 다시 반대 방향으로 이동하여 window 객체에 도착한 다음 이벤트 소멸

이벤트가 흘러가는 과정을 캡쳐링 단계와 버블링 단계로 구분

캡쳐링(capturing) 단계 : 이벤트가 window 객체에서 타겟 객체(이벤트가 발생된 태그) 까지 모든 객체에 이벤트가 전달되는 과정 - 이벤트가 거쳐간 객체에 설정된 이벤트 처리 함수가 호출

버블링(Bubbling) 단계 : 이벤트가 타겟 객체(이벤트가 발생된 태그)로 부터 window 객체까지 모든 객체에 이벤트가 전달되는 과정 - 이벤트가 거쳐간 객체에 설정된 이벤트 처리 함수가 호출
